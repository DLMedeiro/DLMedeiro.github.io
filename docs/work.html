Job Finder

JobFinder is a full-stack web application built with React, Node.js, Express, and PostgreSQL.

This application utilizes a custom RESTful job searching API, a PostgreSQL database deployed through ElephantSQL, and mock data generated with Mockaroo.

With a Mobile-first design, JobFinder simplifies the job search process, offering a user-friendly interface and providing a seamless and responsive experience for both job seekers and employers across a variety of devices.

War Card Game

Responsive full stack application that recreates the classic card game War.

Utilizes React state management to dynamically adapt data based on user interactions, ensuring seamless gameplay and provides a smooth and interactive user experience.

Integration the Deck of Cards API retrieves and incorporates the necessary card data, enabling accurate representation of the game.

Game Play: Players flip a single card from their hand, placing it face up in the center of the table. The player with the higher card wins and collects both cards, returning the cards to their personal deck. If players flip the same card, then players go to war.
